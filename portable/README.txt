                     المتطلبات

قبل التشغيل، تحتاج تنزيل:

1. Python 3.11 Embedded:
   https://www.python.org/ftp/python/3.11.9/python-3.11.9-embed-amd64.zip
   → فك الضغط في مجلد: python/

2. MongoDB 6.0:
   https://fastdl.mongodb.org/windows/mongodb-windows-x86_64-6.0.13.zip
   → فك الضغط في مجلد: mongodb/
   → أنشئ مجلد: mongodb/data/

═══════════════════════════════════════════════════════════════
                   التثبيت (مرة واحدة)
═══════════════════════════════════════════════════════════════

1. افتح Command Prompt في مجلد MoneyTransfer

2. فعّل pip في Python Embedded:
   - افتح ملف python/python311._pth بالمفكرة
   - أزل # من سطر: #import site
   - احفظ الملف

3. ثبّت pip:
   python\python.exe -c "import urllib.request; urllib.request.urlretrieve('https://bootstrap.pypa.io/get-pip.py', 'get-pip.py')"
   python\python.exe get-pip.py

4. ثبّت المكتبات:
   python\python.exe -m pip install -r backend\requirements.txt
